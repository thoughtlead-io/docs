Join us weekly for **Office Hours** to discuss all things Thoughtly. [RSVP here](https://thoughtly.com/office-hours) to be notified about upcoming sessions.

#### Calendarly & GHL Integration Updates [00:02:00]

- Discussed recent updates to Calendarly integration, which should now work correctly with scheduling questions.
- V1.5 includes updates to support scheduling with GHL integrated workflows.

#### Outbound Call Delay and Voicemail Issues [00:03:00]

- Addressed problems with calls starting immediately upon connection, missing initial conversation parts.
- Proposal to add a delay or a slider to manage start timing and accommodate greetings.
- Voicemail system improvements were discussed, with a potential shift to Telnix for better detection.

#### V1.5 Builder Feedback and Automation Challenges [00:10:00]

- Concerns about V1.5 builder's rigid structure, compared to fluidity desired in AI conversation agents.
- Desire for V1.0 builder's flexibility with actions in speak nodes, especially for advanced automation and API integration.

#### V1.5 vs. V2 Builder Plans [00:15:00]

- V1.5 was designed for structured, compliance-driven call flows, useful for some enterprise needs.
- V2 will introduce separate models: V2 linear for strict call flows and V2 conversational for flexible, AI-driven interactions.
- Anticipation for the forthcoming V2 builder that integrates actions across nodes.

#### Agent Use in Car Dealerships [00:20:00]

- Discussion on how Thoughtly could optimize operations in car dealerships.
- Introduction to a potential partner network offering managed service setups.
- Pricing strategies and partnership opportunities in assisting beginners.

#### Creating Contacts and Texting with Agents [00:25:00]

- Creation of contacts during calls is possible but not recommended due to time constraints.
- Suggested post-call automation for contact creation with custom data capture.
- Texting workflows require setting unique agents per interaction type to ensure optimal workflow handling.

#### Automation and CRM Integration [00:35:00]

- Insights on potential powerful uses of the platform through automation, especially with external CRMs.
- Mention of future integration aims, including Meta Lead Ads and asynchronous connections like BooSpace and Google Sheets.

#### Leveraging Advanced Memory [00:45:00]

- Capability to account for previous interactions using metadata and API calls.
- Importance of letting users customize the interaction profile for improved call handling and AI effectiveness.

#### Feedback on Appointment Setting Outcomes [00:55:00]

- Issues raised with viewing appointment outcomes in V1.5, recommendations for UI enhancements.
- Hover feature suggested as a potential solution for clarity in user interfaces.

#### Partner Program and Community Contributions [01:00:00]

- Encouragement for active community participation and information exchange in Slack channels.
- Partner programs and client onboarding as a growth opportunity for experienced users.

#### Texting Feature and Caller ID Setup [01:05:00]

- Recent texting feature integration feedback and potential dual communication channels.
- Clarifications requested on caller ID settings for number handling.

#### Final Community Building Advice [01:10:00]

- Recommended steps for troubleshooting verification processes.
- Building local agent setups for verification and facilitating multi-purpose number handling.
